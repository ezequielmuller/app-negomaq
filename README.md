# 🔪 NegoMaq App

Sistema de vendas de facas desenvolvido com **Quasar Framework**, **Vue 3**, **TypeScript** e **Composition API**.

## 🌐 Acesso Online

O aplicativo está hospedado na **Vercel** e está disponível para acesso imediato. 

📚 **Link para acesso: https://negomaq.vercel.app/home**

Acesse a documentação completa da API no Postman:

📚 **[Documentação da API - Postman](https://documenter.getpostman.com/view/42551640/2sB3WpSMJN)**

## 🛠️ Tecnologias

- [Quasar Framework](https://quasar.dev/) - Framework Vue para aplicações cross-platform
- [Vue 3](https://vuejs.org/) - Framework JavaScript reativo
- [TypeScript](https://www.typescriptlang.org/) - Tipagem estática para JavaScript
- [Composition API](https://vuejs.org/guide/extras/composition-api-faq.html) - Abordagem moderna do Vue 3

## 📋 Requisitos

- Node.js 16+ 
- npm ou yarn
- Conhecimento básico de Vue 3 e TypeScript

## 📦 Instalação

Clone o repositório e instale as dependências:

```bash
git clone <seu-repositorio>
cd app-negomaq
yarn
# ou
npm install
```

## 🚀 Desenvolvimento

Execute o aplicativo em modo de desenvolvimento com hot-reload:

```bash
quasar dev
```

O aplicativo estará disponível em `http://localhost:8080` por padrão.

## 🔧 Comandos Disponíveis

### Lint
Verificar e reportar erros de código:

```bash
yarn lint
# ou
npm run lint
```

### Format
Formatar automaticamente o código:

```bash
yarn format
# ou
npm run format
```

### Build (Produção)
Compilar o projeto para produção:

```bash
quasar build
```

Os arquivos compilados estarão em `dist/spa/`.
